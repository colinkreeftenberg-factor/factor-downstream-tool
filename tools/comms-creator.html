Welcome at Factor's communication creator. Create customer comms in a few easy steps.
01 Upload the customer list
At least upload customer_id, locale and market. Merge tags are optional.
Download an example file here to upload, or use it as a starting point: template file
02 Write the message, per locale
Each locale gets its own subject line, preheader and content. Switch tabs to move between locales — nothing you've written is lost. A dot on a tab means that locale still needs a subject and content before it can be exported.
03 Review
Checks every locale for missing content and merge tags with no value in your file, and shows a live preview merged with a real row from your list.
04 Download
One unicode text file per locale, tab-separated, UTF-8. Ready to upload over FileZilla.
