Made & maintained by Colin Kreeftenberg — questions, edits or new markets go to him.
Use this tool to strategically plan swaps and shortages into the Box Files. It will spread shorts and swaps instead of clustering it to one group of customers. You can select substitutes based on protein- and allergen requirements.
No zone or meal-choice data yet, so impact spreads evenly across the run instead. Protein/allergens come from the Meal Database when a recipe's Rackfile code matches — otherwise it's a flagged guess. Compensation is automatic: €5 / 120 kr swaps by market, price-based shortages (manual fallback until SE/DK box prices land).
